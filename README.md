# CTF Writeups

В этом репозитории собраны мои райтапы CTF-заданий и учебных лабораторных работ по информационной безопасности.

Цель проекта — фиксировать процесс обучения, объяснять ход мыслей при решении задач и делать сложные темы более понятными для новичков.

## Содержание
- Web
- Networking
- OSINT
- Linux
- Misc

## Примечания
- Основной упор сделан на логику решения, а не только на команды
- Все задания решались в легальных учебных средах (CTF, лаборатории, тренировочные платформы)
- Репозиторий создан исключительно в образовательных целях

## Статус
Репозиторий активно пополняется по мере решения новых заданий.
